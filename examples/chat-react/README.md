# chat-react — Vite + React 18 demo

```bash
pnpm install
pnpm dev
```

Then open the URL Vite prints. Files of interest:

- [`src/App.tsx`](./src/App.tsx) — `renderChatMarkdown` wired to a chat bubble component
- [`src/styles.css`](./src/styles.css) — bubble layout

The chat-markdown surface is identical to the [vanilla](../chat-vanilla/) and
[Svelte](../chat-svelte/) demos — only the host framework differs.
